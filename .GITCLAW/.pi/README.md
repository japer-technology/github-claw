Agent configuration and personality system for gitclaw. This folder defines who the agent is, how it thinks, and what it knows how to do.

## Contents

| File / Folder | Purpose |
|---------------|---------|
| `settings.json` | LLM provider, model, and thinking level configuration |
| `APPEND_SYSTEM.md` | System prompt appended to every session |
| `BOOTSTRAP.md` | Instructions for hatching a new agent identity |
| `skills/` | Modular skill packages that extend agent capabilities |
